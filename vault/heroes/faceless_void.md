# Faceless Void
> [2026-05-28]: ROLE -- Position 1 carry.
> [2026-05-28]: IDENTITY -- Considered the best late-game hero in the game. Chronosphere is one of the strongest teamfight ultimates available -- freezes all units in a large AOE while Void moves freely inside it.
> [2026-05-28]: LANING -- Struggles in lane unless extracting value from Time Dilation. Not a strong early laner -- requires a supportive lane setup.
> [2026-05-28]: MID GAME -- Lacks damage in the mid game before items complete. Usually paired with a high-damage hero to maximise Chronosphere value -- the Chronosphere itself does not deal damage, so an ally inside it is needed to convert the lockdown into kills.
> [2026-05-28]: INVOKER COMBO -- Very common pairing with Invoker. Chronosphere + Invoker spell combo (Tornado, Meteor, Cold Snap, Ice Wall) inside the Sphere is one of the highest damage combos available. Both heroes scale into the late game together.
> [2026-05-28]: META (DreamLeague S29) -- 37.5% win rate on 16 picks, 31 bans. Below-average win rate -- difficult late-game hero to pilot and get value from in all matchups.
